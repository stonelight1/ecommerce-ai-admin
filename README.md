# ecommerce-ai-admin

English | 中文

A static admin dashboard prototype for ecommerce operations, covering reporting, inventory, after-sales, finance, data sync, and AI assistant modules.

一个面向电商业务运营的静态管理后台原型，涵盖运营报表、采购库存、客服售后、财务导出、数据同步和 AI 助手等核心模块。

---

## Overview | 项目简介

**ecommerce-ai-admin** is a frontend static prototype designed for ecommerce business operations and AI automation scenarios.  
It focuses on building a modern internal admin system for teams such as operations, procurement, customer service, and finance.

**ecommerce-ai-admin** 是一个面向电商运营与 AI 自动化场景的前端静态原型项目。  
它主要用于构建现代化的企业内部管理后台，适合运营、采购、客服、财务等团队使用。

This project currently provides a static UI prototype and can be used as a starting point for future product design, frontend development, and backend integration.

当前项目提供的是静态 UI 原型，可作为后续产品设计、前端开发和后端对接的起点。
